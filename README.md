# git-cleanup

**Work in progress**

## Installation

**Homebrew (macOS)**

```
brew tap josa42/homebrew-git-tools
brew install git-cleanup
```

**Other**

```
go get github.com/josa42/git-cleanup
```

## Usage

```
Usage:
  git-cleanup branches
  git-cleanup keep     [--all]

Options:
  -h --help          Show this screen.
  --version          Show version.
```
